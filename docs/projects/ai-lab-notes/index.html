[샘플 프로젝트] AI Lab Notes

AI와 개발 학습 과정을 기록하는 개인 블로그 & 포트폴리오 사이트. Astro + Tailwind CSS로 빌드, GitHub Pages로 배포.

GitHub →

프로젝트 소개

이 사이트 자체가 하나의 프로젝트입니다. AI와 개발을 공부하면서 배운 것들을 기록하고, 동시에 포트폴리오로도 활용하기 위해 만들었습니다.

기술 선택 이유

  • Astro: 콘텐츠 중심 사이트에 최적화. 빌드 시 정적 HTML로 변환되어 빠른 로딩.
  • Tailwind CSS: 컴포넌트 없이도 일관된 디자인 유지 가능.
  • GitHub Pages: 무료 호스팅, GitHub와 완전 통합.

주요 기능

  • Markdown/MDX 기반 블로그 포스트
  • 프로젝트 포트폴리오 페이지
  • GitHub Actions 자동 빌드 & 배포
  • 반응형 디자인
  • SEO 최적화 (메타태그, sitemap, OG 태그)
